WHAT IT DOES
• Save every custom request, deadline, and order status in one place.
• Get a desktop reminder on the morning of each due date.
• Track progress across 5 stages: Received → Making → Packing → Shipped → Review.

WHO IT'S FOR
Custom-order sellers on Etsy, Shopify, Amazon Handmade, eBay, and any marketplace where buyer requests come with personalization details, sizes, colors, or rush deadlines.

DESKTOP ONLY
Custom Order Notes runs as a Chrome desktop extension. There is no mobile companion app — your data lives on your computer. We recommend doing a JSON backup once a month and importing it on any other PC you use.

MULTI-PC TIP
Run the same shop from two computers? Export a JSON backup on PC1, drop it into PC2's "Restore from backup" — takes 10 seconds, no cloud sync needed. Your previous data is auto-saved before every restore, so accidental clicks can't wipe your work.

WHY LOCAL-FIRST
Your buyer notes never leave your computer. No servers. No syncing. No data breaches. If you want to move to a different PC, download a JSON backup and restore it on the other side — done.

==============================
単一用途の説明
==============================
A local notebook for managing custom order notes with due-date reminders and status tracking, designed for online marketplace sellers.

==============================
storage が必要な理由
==============================
Required to save order notes, due-date schedules, checklists, and user settings locally on the device using chrome.storage.local. No data is sent to any external server.

==============================
alarms が必要な理由
==============================
Required to schedule a single daily check at 9:00 AM that surfaces orders whose due date has arrived or passed, so the seller does not miss a shipping deadline.

==============================
notifications が必要な理由
==============================
Required to display a desktop reminder when an order's due date arrives, allowing the seller to jump back to the order page.

==============================
ホスト権限が必要な理由（content_scripts の matches）
==============================
https://extensionpay.com/*:
Required by ExtensionPay (the payment provider) to detect when a one-time payment is completed. The ExtPay.js content script runs on the ExtensionPay checkout page to communicate payment status back to the extension.

==============================
リモートコードを使用しているか
==============================
No. This extension does not use any remotely hosted code. All scripts are bundled within the extension package.
