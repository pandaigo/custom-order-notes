Privacy Policy

Last updated: 2026-05-07

Custom Order Notes ("the Extension") is a Chrome browser extension that helps online marketplace sellers track custom buyer requests, deadlines, and order statuses entirely on their local device.

Data Collection

The Extension does not collect, transmit, or share any personal data. Specifically:

  • No data is sent to any external server.
  • No analytics, tracking, or telemetry is included.
  • No user accounts are required.
  • No cookies are set by the Extension.

Data Storage

All user data — including order numbers, buyer names, custom requests, due dates, statuses, and checklists — is stored locally on the user's device using the Chrome Storage API (chrome.storage.local). This data never leaves the device unless the user explicitly exports it as a CSV or JSON file.

Permissions

The Extension requests the following permissions:

  • storage — To save order notes, schedules, and settings locally on the device.
  • alarms — To schedule a single daily check that surfaces orders whose due date has arrived.
  • notifications — To display a desktop reminder when an order is due.

Third-Party Services

The Extension uses ExtensionPay (extensionpay.com) to process one-time payments and verify purchase status. When a user upgrades to the Pro version, payment is handled by ExtensionPay via Stripe. The Extension does not share order notes, buyer names, or any other user-entered data with these services.
